# 🛒 Retail Sales Analysis

An end-to-end data analytics project focused on uncovering sales trends, customer behavior, and regional performance using Python, MySQL, SQL, and Power BI.

## 🚀 Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- MySQL
- SQL
- Power BI

## 📊 Key Features
✔ Data Cleaning & Exploratory Data Analysis  
✔ Database Integration using MySQL  
✔ SQL-based Business Analysis  
✔ Interactive Power BI Dashboard  
✔ Sales, Profit, Customer & Regional Insights

## 📈 Dashboard Highlights
- Total Sales: 20M+
- Total Profit: 5M+
- Profit Margin: 25.09%
- Top-performing products and regions
- Customer and sales trend analysis


## 🎯 Skills Demonstrated
Python • SQL • MySQL • Power BI • EDA • Data Visualization • Business Intelligence
